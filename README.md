# 台灣高中生 7000 單字練習工具

一個專門給台灣高中生練習「7000 單」的線上小工具。  
選擇等級（level）就能開始複習，還有「錯題本」幫你整理常錯單字，隨時補強弱點 💪

---

## 功能介紹

- 🎯 **分級練習（Level Practice）**  
  - 依照等級切好的單字表（如 Level 1、Level 2...）  
  - 選擇 level 後即可開始作答  
  - 題目隨機出題，不會只照表上順序出現

- 📚 **錯題本（Wrong Book）**  
  - 答錯的單字會自動收進錯題本  
  - 可以單獨打開「錯題練習模式」  
  - 幫你針對常錯單字加強記憶

- 📈 **練習進度顯示**  
  - 顯示目前該 level 已練習幾題、答對率等資訊  
  - 方便掌握自己每個等級的熟悉度

---

## 使用方式

1. 打開網頁 [voc.justhandsome.me](https://voc.justhandsome.me)
2. 在畫面上方的 **「Choose level」** 下拉選單選擇想練習的等級。
3. 點選 **「Start / Resume」** 開始或繼續該 level 的練習。
4. 作答過程中若有寫錯，單字會自動被記到 **錯題本**。
5. 想針對錯題加強時，切換到錯題本模式，再次練習這些單字。
